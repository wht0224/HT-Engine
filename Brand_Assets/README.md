# HT_Engine Brand Assets

This folder contains brand assets for HT_Engine. Developers using HT_Engine must display proper attribution in their games/applications.

## Attribution Requirements

All games and applications developed using HT_Engine **must** display the following attribution during startup:

### Required
- **Text:** "Made with HT_Engine" or "Powered by HT_Engine"
- **Duration:** Minimum 2 seconds
- **Location:** Clearly visible (splash screen, loading screen, or main menu)

### Optional
- HT_Engine logo (when available)
- Link to HT_Engine GitHub repository

## Available Assets

### Logo
- `logo.png` - HT_Engine logo (when available)
- `logo_white.png` - White version for dark backgrounds (when available)
- `logo_black.png` - Black version for light backgrounds (when available)

### Splash Video
- `splash.mp4` - Official HT_Engine splash video (when available)
- `splash.webm` - WebM format for web games (when available)

## Usage Examples

### Text Only
```
Made with HT_Engine
```

### With Logo
```
[HT_Engine Logo]
Powered by HT_Engine
```

### In Game Code (Example)
```python
# Display splash screen for 2 seconds
show_splash_screen("Made with HT_Engine", duration=2.0)
```
